## react-freeze-body

Add `overflow: hidden;` to the `document.body`, the react way.

<br />

## Install

```
$ npm install react-freeze-body --save
```

## Usage

```js
import ReactFreezeBody from 'react-freeze-body'

render () {
  return (
    <ReactFreezeBody />
  )
}
```
